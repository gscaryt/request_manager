Settings — your working library

This is your layer on top of the active library. Tick preselected to start a request selected on every reset. Tag EA · Editor to choose which checklist tabs show it. Click a request’s name to adapt its title and text for your field (system text can always be restored); use × to hide it. Your own one-off additions are custom requests. To change the shared foundation itself, use Setup.

🔒 Read-only. Unlock Dev mode (top bar) to change your preselected set or adapt request text.

Library Setup — the shared foundation

Build the journal-wide standard a chief editor hands to every editor: pick a Master Library, set its sections, and write its system requests. These are not personal customisations — for those, use Settings. Start a new library from 0 to stay journal-agnostic.

🔒 Read-only. Unlock Dev mode (top bar) to author libraries.
